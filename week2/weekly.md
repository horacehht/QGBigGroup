问了问朋友，**说是(3，)和(3,1)导致的错误**，要在引入时对输入y处理才不会发生这样的情况。

于是main.py中导入数据的代码更改为：

```python
data = load_breast_cancer()
X = data['data']
Y = data['target'].reshape(-1, 1)  # 防一个奇怪的Bug
```
